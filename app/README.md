# Ambient 101

Vite + React + TypeScript + Tone.js + Tonal.

Product and workshop scope: [`../docs/design.md`](../docs/design.md).

**Phase 1:** Start audio, then play two demo tape loops (7s and 11s) with independent transport timing. Loops appear as horizontal rows with reel + mini melody playheads.

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```
